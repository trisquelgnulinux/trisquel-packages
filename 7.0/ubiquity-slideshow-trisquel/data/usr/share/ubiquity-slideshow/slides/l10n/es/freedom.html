Bits de libertad

Nuestro proyecto tiene el fuerte compromiso de distribuir solamente software libre, siguiendo las pautas descritas por la Free Software Foundation. Estas pautas extienden la definición de software libre para asegurar que se pueda usar, estudiar, modificar y compartir cada parte y pieza de Trisquel.

También es posible apoyar al software libre convirtiéndose en miembro de la Free Software Foundation. Si lo haces te mandarán un disco USB con forma de tarjeta de socio, cargado con una versión Live de Trisquel.

Prometemos que Trisquel será siempre libre, pero si cometiéramos un error o encuentras algo inapropiado, por favor contacta con nosotros y lo arreglaremos.
