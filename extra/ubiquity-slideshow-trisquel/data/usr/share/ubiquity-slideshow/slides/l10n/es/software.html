Obteniendo nuevo software

Trisquel tiene software preinstalado para las tareas comunes de cada día, y puede extenderse fácilmente con la ayuda de los miles de paquetes disponibles en nuestro repositorio. En un catalogo gráfico fácil de usar, existen paquetes para la educación, ciencia, programación, redes, juegos y mucho más.

Nuestro software se actualiza constantemente tanto por seguridad y arreglo de errores como por mejoras diversas, y estas actualizaciones se aplican fácilmente a través de una sencilla herramienta gráfica. Serás notificado cada vez que haya actualizaciones disponibles, y cuando se publique una nueva versión de Trisquel.

El navegador de internet puede también ser mejorado instalando extensiones libres tales como bloqueadores de publicidad, gestores de descarga, administradores de marcadores, temas visuales...
